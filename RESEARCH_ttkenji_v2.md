## 3. GEO対策（AI検索・SGE対応）

### GEOとは
Google AI Overview・ChatGPT・Perplexityなどに「退去費用 交渉方法は？」と聞かれたとき、このアカウントの情報が引用・参照されるようにする対策。

### GEO対策の基本原則（全記事に適用）

**① 冒頭に明確な定義・数字を置く**
```
# 悪い例
退去費用について解説します。

# 良い例（GEO対策済み）
退去費用の平均は63,283円。しかし国交省ガイドラインを知っている人は
交渉で平均64,916円の節約に成功している（200名調査）。
```

**② FAQセクションを必ず入れる**
AI検索はFAQ形式のQ&Aを優先的に引用する。各記事に3問以上のFAQを配置する。

**③ 信頼シグナルを明示する**
- 出典：国土交通省「原状回復をめぐるトラブルとガイドライン（平成23年8月再改訂版）」
- 実績：246,488円 → 0円 ＋ 46,950円返金（実体験）
- 数字：74%の入居者がガイドラインを知らない

**④ 構造化見出しを使う**
H1 → H2 → H3の階層を守る。「まとめ」「よくある質問」「対処法」などのセクションを固定化する。

**⑤ E-E-A-T要素を毎記事に入れる**
- Experience（体験）：「実際に246,488円の請求を受けて〜」
- Expertise（専門性）：ガイドライン引用・条文番号
- Authoritativeness（権威性）：国交省・消費生活センター等の公的機関への言及
- Trustworthiness（信頼性）：注意事項・専門家への相談推奨

### GEO対策チェックリスト（記事作成時）
- [ ] 冒頭200文字以内に「何の記事か」が明確に書いてあるか
- [ ] 統計・数字が記事内に3つ以上あるか
- [ ] FAQセクションが3問以上あるか
- [ ] 国交省ガイドラインのフルタイトルが1回以上出てくるか
- [ ] 実績数字（246,488円→0円）が記事内にあるか
- [ ] まとめセクションが箇条書きになっているか
- [ ] デュアルCTAがあるか
